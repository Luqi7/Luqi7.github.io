搭建个人博客

Posted on 2020-04-13

1.安装nodejs

在命令行下操作

  • node -v 查看node版本

  • npm -v 查看包管理器npm版本

2.利用npm安装cnpm（淘宝镜像）

  • npm install -g cnpm –registry=https://registry.npm.taobao.org

3.利用cnpm安装hexo博客框架

  • cnpm install -g hexo-cli

  • 用 hexo -v 验证是否安装成功

4.使用hexo博客框架

  • 建立一个空文件夹blog，之后操作全都在此文件夹下面进行（使用过程出现错误可以直接删掉该文件夹重头再来）

    • 初始化一个博客 hexo init

    • 启动博客 hexo s (start)，平时写完内容后用来做预览

    • 写博客 hexo n（new）

    • 清理缓存 hexo clean

    • 生成内容 hexo g（generate）
